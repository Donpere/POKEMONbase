# POKEMON BASE
A little exercise for my porfolio. I found this API for my son on the net. First add a different Color Setting, then add a random pokemon each time you refresh the page, then a little module to calculate the weight of each request on API... Working for now on the local storage or session storage to make less requests on API.

I have several others functions in my mind, just to test some new skills.

Enjoy it guys !
